= 贡献者公约 - 行为准则

== 我们的承诺
为了营造一个开放和友好的环境，我们作为贡献者和维护者承诺：无论年龄、体型、残障、种族、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向如何，参与我们社区的活动都将不受骚扰。

== 行为标准

=== 积极行为示例：
* 使用友好和包容的语言
* 尊重不同的观点和经验
* 优雅地接受建设性批评
* 关注对社区最有利的事情
* 对其他社区成员表示同理心

=== 不可接受的行为：
* 使用性化的语言或图像，以及不受欢迎的性关注
* 挑衅、侮辱、贬低性评论，以及人身或政治攻击
* 公开或私下骚扰
* 未经许可发布他人信息（如物理或电子邮件地址）
* 在专业环境中明显不合适的其他行为

== 执行责任

社区维护者负责澄清和执行可接受行为的标准，并将对任何被认为不适当、威胁、冒犯或有害的行为采取适当和公平的纠正措施。

== 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共场合代表社区时。

== 执行

虐待、骚扰或其他不可接受的行为可通过 [联系方式] 向社区维护者报告。所有投诉都将得到审查和调查，并将产生被认为必要和适当的回应。

== 执行指南

社区维护者将遵循以下社区影响指南来确定对任何违规行为的处罚：

. *纠正* - 纠正不当行为，公开或私下道歉
. *警告* - 针对持续行为的警告
. *临时封禁* - 暂时禁止参与
. *永久封禁* - 永久禁止参与
