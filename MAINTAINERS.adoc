== 🎯 成为维护者
我们欢迎活跃的贡献者加入维护团队！

=== 要求
* 持续贡献 3 个月以上
* 合并至少 5 个有意义的 PR
* 积极参与社区讨论
* 理解项目愿景和架构

=== 申请方式
. 通过持续贡献展示你的能力
. 联系现有维护者表达意向
. 维护团队讨论决定
. 公开宣布新维护者

== 📜 决策流程

=== 技术决策
* 小改动：单个维护者批准
* 中等改动：2 个维护者批准
* 重大改动：维护团队投票 (简单多数)

=== 社区决策
* 通过 GitHub Discussions 征求意见
* 重要变更需要社区公示期 (至少 7 天)
