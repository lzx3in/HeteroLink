= 图标文件

== 系统主题图标（主要）

大部分 UI 图标使用 Qt 系统主题图标，无需本地文件。

[cols="2*",options="header"]
|===
|功能 |图标名

|应用
|`applications-engineering`

|连接
|`network-connect`

|断开
|`network-disconnect`

|开始
|`media-playback-start`

|停止
|`media-playback-stop`

|保存
|`document-save`

|打开
|`document-open`

|导出
|`document-export`

|设置
|`preferences-system`

|帮助
|`help-contents`

|关于
|`help-about`

|添加
|`list-add`

|移除
|`list-remove`

|全屏
|`view-fullscreen`

|退出
|`application-exit`
|===

== 自定义图标（本地 SVG）

以下图标系统主题没有，使用本地 SVG：

* `device_online.svg` - 在线设备状态（绿色圆点）
* `device_offline.svg` - 离线设备状态（红色圆点 + 斜线）

== 可选：替换为精美图标

如需更美观的图标，可从以下来源获取：

* https://pictogrammers.com/library/mdi/[Material Design Icons]
* https://github.com/microsoft/fluentui-system-icons[Fluent UI System Icons]
* https://tabler.io/icons[Tabler Icons]

下载后替换 `resources.qrc` 中的引用并转换为 PNG/SVG。
