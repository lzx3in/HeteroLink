=== 更新文档

发现文档问题？欢迎改进：

. 点击页面右上角的 "Edit this page"
. 提交修改

=== 文档规范

* 使用 AsciiDoc 格式
* 代码示例添加语法高亮
* 图片放在 `images/` 目录
* 保持中英文术语一致

=== 构建文档

[source,bash]
----
# 安装依赖
pip install mkdocs mkdocs-material

# 本地预览
cd docs
mkdocs serve

# 构建静态站点
mkdocs build
----
