# 快速开始系列

欢迎使用 HeteroLink！这个系列将带你快速上手。

---

## 📖 阅读顺序

### 第一步：环境准备

在开始之前，确保你的开发环境已就绪：

- [安装指南](installation.md) - 安装 ESP-IDF 和工具链

### 第二步：快速入门

完成环境准备后，跟随快速入门教程：

- [10 分钟上手](quick-start.md) - 编译、烧录、测试

### 第三步：深入学习

掌握基础后，探索更多功能：

- [架构设计](../firmware/architecture.md) - 理解系统架构
- [示例项目](../examples/README.md) - 实践练习
- [问题排查](troubleshooting.md) - 常见问题解决

---

## 🎯 学习路径

```
新手 → 安装环境 → 快速上手 → 运行示例 → 开发自己的项目
```

### 预计时间

| 阶段 | 内容 | 时间 |
|------|------|------|
| 环境准备 | 安装 ESP-IDF、配置工具链 | 30 分钟 |
| 快速上手 | 编译、烧录、测试 | 15 分钟 |
| 示例练习 | 运行 2-3 个示例 | 1 小时 |
| 项目开发 | 开发自己的应用 | 视项目而定 |

---

## 📋 前提条件

### 硬件要求

- ESP32 开发板 (ESP32-C6 推荐)
- USB 转串口适配器
- 电脑 (Linux/macOS/Windows)

### 软件要求

- Git
- Python 3.8+
- 至少 10GB 可用磁盘空间

### 知识要求

- 基础 C/C++ 编程
- 嵌入式系统概念
- 命令行基础
