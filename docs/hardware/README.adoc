== 🔧 生产指南

=== PCB 制造

. 下载 Gerber 文件
. 提交到 PCB 制造商
. 选择参数：
* 板材：FR-4
* 层数：2 层
* 板厚：1.6mm
* 铜厚：1oz
* 阻焊颜色：绿色/黑色/蓝色

=== 元器件采购

参考 BOM 表采购元器件。

推荐供应商：
* Digi-Key
* Mouser
* LCSC (立创商城)
* 得捷电子

=== SMT 贴片

. 提供 Gerber + BOM + 坐标文件
. 选择元器件面 (Top/Bottom)
. 指定特殊工艺 (如阻抗控制)

=== 手工焊接

对于原型制作，可以手工焊接：
* 使用恒温焊台
* 先焊接低矮元件
* 最后焊接连接器
