# HeteroLink 子板

ESP32 协处理器子板 - HeteroLink 的核心处理单元。

---

## 📋 产品概述

HeteroLink Subboard 是一款基于 ESP32 系列芯片的协处理器板卡，为单板计算机、微控制器和嵌入式系统提供强大的调试、监控和数据采集能力。

---

## ✨ 核心特性

### 通信能力

- **WiFi 6 (802.11ax)** - 高速无线连接
- **Bluetooth 5 (LE)** - 低功耗蓝牙
- **UART × 2** - 串口通信
- **SPI × 2** - 高速同步通信
- **I2C × 1** - 传感器接口
- **GPIO × 12** - 通用输入输出

### 采集能力

- **ADC 12-bit × 6** - 模拟信号采集
- **GPIO 探测** - 数字逻辑分析
- **采样率**: 最高 100 kS/s
- **输入范围**: 0-3.3V

### 处理能力

- **CPU**: RISC-V 单核 160 MHz
- **Flash**: 4MB (可定制)
- **PSRAM**: 可选 2MB
- **RTOS**: FreeRTOS

## 🔌 接口定义

### USB-C 接口

```
┌─────────────────┐
│   USB Type-C    │
│  ┌───────────┐  │
│  │ ○ ○ ○ ○   │  │  CC1/CC2: 配置通道
│  │ ○ ○ ○ ○   │  │  D+/D-: USB 2.0
│  └───────────┘  │  VBUS: 5V 电源
└─────────────────┘  GND: 地
```

### GPIO 扩展接口

| 引脚 | 功能 | 备注 |
|------|------|------|
| GPIO0 | BOOT 模式 | 启动配置 |
| GPIO1 | UART0 TX | 调试输出 |
| GPIO2 | UART0 RX | 调试输入 |
| GPIO3-8 | SPI + DMA | 高速数据传输 |
| GPIO9-10 | UART1 | 近端通信 |
| GPIO11-12 | I2C | 传感器接口 |
| GPIO13-18 | ADC/GPIO | 模拟/数字探测 |

### 电源接口

```
┌──────────────┐
│  Power Jack  │
│  ┌────────┐  │
│  │ +5V   │  │  输入：5V DC
│  │ GND   │  │  电流：500mA (推荐)
│  └────────┘  │
└──────────────┘
```
