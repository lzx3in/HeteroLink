## 📋 概述

上位机是 HeteroLink 系统的核心控制和数据可视化平台，提供跨平台（Windows/Linux/macOS）的图形界面应用。

### 核心定位

- **设备管理**: 发现、连接、配置多个 ESP32 协处理器子板
- **数据采集**: 实时接收和展示 kHz 级遥测数据
- **控制指令**: 下发配置、命令到设备
- **数据记录**: 持久化存储采集的数据
- **可视化**: 波形、图表、仪表盘等可视化展示
- **告警系统**: 阈值监控、异常通知

## 🎯 功能模块

### 1. 近端通道 (UART)

**功能**:
- 自动扫描可用串口
- 手动配置串口参数（波特率、数据位、校验位等）
- 实时数据流接收（kHz 级）
- 指令下发（配置、控制命令）
- 连接状态监控

**技术要点**:
- 使用 QSerialPort 进行串口通信
- 环形缓冲区处理高速数据流
- 异步非阻塞 I/O
- 自动重连机制

### 2. 远端通道 (MQTT)

**功能**:
- 连接 MQTT Broker（支持 TLS）
- 订阅设备状态 Topic
- 发布控制命令
- Last Will 遗嘱消息处理
- 多设备集群管理

**技术要点**:
- 使用 QMqtt（Qt 官方 MQTT 库）或 Paho MQTT C++
- 支持 MQTT 5.0 特性
- 连接池管理
- 离线消息缓存

### 3. 数据可视化

**功能**:
- 实时波形图（多通道）
- 数据表格（可排序、过滤）
- 仪表盘（数值、进度条）
- 散点图、柱状图
- 数据导出（CSV、JSON）

**技术要点**:
- 使用 QCustomPlot 或 Qt Charts
- GPU 加速渲染
- 数据降采样（大数据量时）
- 可配置的时间窗口

### 4. 设备管理

**功能**:
- 设备发现（自动/手动）
- 设备分组
- 设备配置保存
- 固件升级（OTA）
- 设备状态监控

### 5. 数据记录

**功能**:
- 自动/手动开始录制
- 分段录制（按时间/大小）
- 数据回放
- 导出为标准格式

### 6. 告警系统

**功能**:
- 阈值配置（上限、下限）
- 告警触发（声音、弹窗、通知）
- 告警历史
- 告警导出

## 📐 协议设计

### UART 二进制协议

**帧格式**:
```
┌─────────┬─────────┬──────────┬──────────┬─────────┬──────────┐
│ 帧头    │ 设备 ID  │ 命令字    │ 数据长度  │ 数据     │ 校验和   │
│ 2 bytes │ 1 byte  │ 1 byte   │ 2 bytes  │ N bytes │ 2 bytes  │
│ 0xAA55  │ 0x00-0xFF│ 0x00-0xFF│ 0x0000-0xFFFF│  payload │ CRC16   │
└─────────┴─────────┴──────────┴──────────┴─────────┴──────────┘
```

**命令字定义**:
| 命令字 | 名称 | 方向 | 描述 |
|--------|------|------|------|
| 0x01 | CMD_HEARTBEAT | 双向 | 心跳包 |
| 0x02 | CMD_CONFIG_REQ | 主机→设备 | 请求配置 |
| 0x03 | CMD_CONFIG_RSP | 设备→主机 | 配置响应 |
| 0x10 | CMD_TELEMETRY | 设备→主机 | 遥测数据 |
| 0x20 | CMD_CONTROL | 主机→设备 | 控制命令 |
| 0x30 | CMD_LOG_UPLOAD | 设备→主机 | 日志上传 |
| 0xFF | CMD_ERROR | 设备→主机 | 错误报告 |

**遥测数据格式** (CMD_TELEMETRY):
```
┌──────────┬───────────┬───────────┬───────────┬───────────┐
│ 时间戳   │ 通道 1     │ 通道 2     │ 通道 3     │ ...       │
│ 4 bytes  │ 4 bytes   │ 4 bytes   │ 4 bytes   │ N bytes   │
│ uint32   │ float32   │ float32   │ float32   │ float32   │
└──────────┴───────────┴───────────┴───────────┴───────────┘
```

### MQTT Topic 规范

**设备状态**:
- 发布：`heterolink/subboard/{device_id}/status`
- 内容：`{"state": "online"|"offline", "timestamp": 1234567890}`

**设备遥测**:
- 发布：`heterolink/subboard/{device_id}/telemetry`
- 内容：`{"ts": 1234567890, "ch1": 1.23, "ch2": 4.56, ...}`

**控制命令**:
- 订阅：`heterolink/subboard/{device_id}/command`
- 内容：`{"cmd": "config"|"control"|"ota", "payload": {...}}`

**告警通知**:
- 发布：`heterolink/subboard/{device_id}/alarm`
- 内容：`{"level": "warning"|"critical", "message": "...", "value": 123.45}`
