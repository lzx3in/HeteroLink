== 🚀 开始使用

=== 1. 克隆项目
[source,bash]
----
git clone https://github.com/HeteroLink/HeteroLink.git
cd HeteroLink/examples
----

=== 2. 选择示例
[source,bash]
----
cd basic-telemetry  # 或其他示例目录
----

=== 3. 编译烧录
[source,bash]
----
# 设置目标芯片
idf.py set-target esp32c6

# 配置项目
idf.py menuconfig

# 编译
idf.py build

# 烧录
idf.py -p /dev/ttyUSB0 flash

# 监视输出
idf.py -p /dev/ttyUSB0 monitor
----
