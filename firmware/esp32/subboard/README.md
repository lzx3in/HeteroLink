# HeteroLink ESP32 Subboard

基于 ESP32 的异构协处理器子板固件。

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                    上位机 (Qt/C++ 跨平台)                    │
│  ┌──────────────┐  ┌──────────────┐  ┌─────────────────┐   │
│  │ 近端通道      │  │ 远端通道      │  │ 数据操作/告警    │   │
│  │ UART 二进制   │  │ MQTT over WiFi│  │ 日志记录        │   │
│  │ kHz 级遥测    │  │ 云端 Broker   │  │ 集群管理        │   │
│  └──────────────┘  └──────────────┘  └─────────────────┘   │
└─────────────────────────────────────────────────────────────┘
                            ↕ (近端 UART / 远端 WiFi)
┌─────────────────────────────────────────────────────────────┐
│              下位机 (ESP32 协处理器子板)                      │
│  ┌──────────────┐  ┌──────────────┐  ┌─────────────────┐   │
│  │ SPI+DMA      │  │ GPIO 数字探测  │  │ ADC 模拟采集     │   │
│  │ 共享内存      │  │ 双模点测      │  │ 零代码即插即用   │   │
│  └──────────────┘  └──────────────┘  └─────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

## 核心功能

| 通道 | 协议 | 功能 | 性能 |
|------|------|------|------|
| 近端 | UART 自定义二进制 | 高频遥测、指令下发 | kHz 级数据上传，μs 级响应 |
| 远端 | MQTT over WiFi | 设备状态上云、远程控制、告警推送 | 集群管理 |
| 板间 | SPI+DMA+ 共享内存 | 高速双向数据传输 | 高速 |
| 点测 | GPIO+ADC | 数字逻辑探测 + 模拟信号采集 | 零代码即插即用 |

## 快速开始

### 环境要求

- ESP-IDF v5.3+
- ESP32 / ESP32-S3 / ESP32-C3 / ESP32-C6

### 编译烧录

```bash
# 设置目标芯片 (根据实际硬件选择)
idf.py set-target esp32c6    # ESP32-C6 (WiFi 6)
# 或：idf.py set-target esp32s3
# 或：idf.py set-target esp32
# 或：idf.py set-target esp32c3

# 配置项目 (WiFi SSID/Password, MQTT Broker)
idf.py menuconfig

# 编译
idf.py build

# 烧录 (替换为实际端口)
idf.py -p /dev/ttyUSB0 flash monitor
```
