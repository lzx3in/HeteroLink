# 占位图标文件

实际项目中需要添加以下 PNG 图标文件：

- app.png (32x32) - 应用程序图标
- connect.png (24x24) - 连接图标
- disconnect.png (24x24) - 断开图标
- start.png (24x24) - 开始采集图标
- stop.png (24x24) - 停止采集图标
- settings.png (24x24) - 设置图标
- device.png (24x24) - 设备图标
- device_online.png (24x24) - 在线设备图标
- device_offline.png (24x24) - 离线设备图标
- alarm.png (24x24) - 告警图标

可以使用 Qt Resource Editor 或从图标库（如 Material Design Icons）获取。

临时解决方案：创建空的 PNG 文件或移除 resources.qrc 中的引用。
